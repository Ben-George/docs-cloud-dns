Prerequisites
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To start using the API and complete the examples in this section, you need the following 
items:

- Rackspace Cloud account. If you don't have one, `sign up`_ for one.
- :ref:`Command line tool or browser client<send-api-requests>` for communicating with the 
  API service
- Configured environment variables for :ref:`cURL <configure-environment-variables>` 
  command line tool.



You can find your account number and API key on the Account Settings page in the Cloud 
Control Panel.
